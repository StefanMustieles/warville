Vickers Crossley Model 1925 (Type 87 Dowa)

  • Year: 1935.
  • Numbers Produced: 89.
  • Crew: 4 (Commander, Gunner, Driver & Assistant Driver).
  • Main Armament: 1 x .50 cal M2HB Heavy Machine Gun.
  • Elevation: -10° to +20°.
  • Turret Traverse: 360° (Manual).
  • Maximum Range: 1.830m.
  • Armour Penetration: 19mm @ 500m @ 30°.
  • Secondary Armament:
    • 1 x .30 cal M1919A4 Machine Gun (AA Mount).
    1 x .30 cal M1919A4 Machine Gun (Coaxial).
    1 x .30 cal M1919A4 Machine Gun (Bow).
  • Ammunition Carried: 1.100 x .50 cal & 6.700 x .30 cal.
  • Height: 2.30m.
  • Width: 2.40m.
  • Length: 4.14m.
  • Combat Weight: 8.523 kg.
  • Ground Clearance: 0.38m.
  • Fording Depth: 1.10m.
  • Trench Crossing: 1.20m.
  • Obstacle Clearance: 0.61m.
  • Climbing Ability: 26°.
  • Radio: SCR 210.
  • Armour:
    • Upper Hull Front: 16mm.
    Lower Hull Front: 16mm.
    Upper Hull Sides: 13mm.
    Lower Hull Sides: 13mm.
    Hull Rear: 6.4mm.
    Hull Top: 6.4mm.
    Hull Bottom: 6.4mm.
    Gun Mantlet: None.
    Turret Front: 16mm.
    Turret Sides: 16mm.
    Turret Rear: 16mm.
    Turret Top: 6.4mm.
  • Maximum Road Range: 118 mph (190 kph).
  • Engine: Continental R-670 (Petrol).
  • Transmission: 5 Forward & 1 Reverse.
  • Maximum Road Range: 120 miles (190 km).
  • Maximum Road Speed: 45 mph (72 kph).
  • Maximum Cross Country Speed: 20 mph (32 kph).
  • Notes; The first Combat Cars were the Cavalry’s way of getting round the restrictions which stated that Tanks were for Infantry use only, so the name Combat Car. This the first model had a D-Shaped turret with no Cupola and was armed with two machine guns.
